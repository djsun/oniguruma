ONIGURUMA FOR RUBY:

Ruby bindings to the Oniguruma regular expression library (no need to recompile Ruby).

FEATURES:

  • Increased performance.
  • Same interface than standard Regexp class (easy transition!).
  • Support for named groups, look-ahead, look-behind, and other cool features!

SYNOPSIS:

 reg = Oniguruma::ORegex.new( '(?<before>.*)(a)(?<after>.*)' )
 match = reg.match( 'terraforming' )
 puts match[0]         <= 'terraforming'
 puts match[:before]   <= 'terr'
 puts match[:after]    <= 'forming'

REQUIREMENTS:

  • Oniguruma library v. 2.0 or greater

INSTALL:

sudo gem install -r oniguruma

BUGS/PROBLEMS/INCOMPATIBILITIES:

  • ORegexp#~ is not implemented.
  • ORegexp#kcode results are not compatible with Regexp.
  • ORegexp options set in the string are not visible, this affects ORegexp#options, ORegexp#to_s, ORegexp#inspect and ORegexp#==.
